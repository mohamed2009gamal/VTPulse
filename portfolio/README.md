# Refactored Portfolio - Clean Architecture

## Structure
- **Frontend**: Feature-based (`src/features/`), external CSS, API services
- **Backend**: MVC (`src/controllers/`, `src/routes/api/`)

## Run
Frontend: `npm start`
Backend: `cd backend && npm start`

## Features
- Auth (OAuth/JWT)
- Admin Dashboard
- Blogs (CRUD, voting, analytics)
- AI Chat
- Contact form
- Visitor tracking

Scalable, production-ready! 🚀
