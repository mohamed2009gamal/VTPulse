# Scroll To Down Button

## Features
- Fixed bottom-center double-arrow SVG from Screenshot-2026-03-18-142412.svg
- Light/dark theme: black/light via CSS var(--text)
- Click: smooth scroll 200px down
- Auto-hide when scrollY >50px (smooth fade/slide)
- Float animation
- Mobile hidden (<=1024px)
- z-index 999
